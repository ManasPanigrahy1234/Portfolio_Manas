### **Changing Colors**
The website uses a beautiful gradient color scheme. To customize:

```css
/* Primary gradient (used throughout) */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* Accent color (golden) */

```

### **Updating Personal Information**
Edit the following sections in `index.html`:

- **Hero Section**: Name, title, and description
- **About Section**: Personal details and statistics
- **Skills**: Programming languages and proficiency levels
- **Projects**: Add your own projects with descriptions
- **Experience**: Update your work history
- **Contact**: Update contact details

### **Adding New Projects**
To add a new project, copy this structure:

```html
<div class="project-card">
    <div class="project-image">
        <i class="fas fa-[icon-name]"></i>
    </div>
    <div class="project-content">
        <h3>Project Name</h3>
        <p>Project description goes here...</p>
        <div class="project-tech">
            <span>Technology 1</span>
            <span>Technology 2</span>
        </div>
        <div class="project-links">
            <a href="#" class="btn btn-small">View Project</a>
        </div>
    </div>
</div>
```

### **Modifying Skills**
Update skill levels by changing the width percentage:

```css
.skill-progress { width: 90%; } /* 90% proficiency */
```
